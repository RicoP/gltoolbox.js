Welcome to GLToolbox.js

This little tool helps me to produce webgl apps and I hope it will also help you to create awesome things.

To add the library to your repro just do the following: 

$ cd your_repo_root/repo_name
$ git submodule add git://github.com/RicoP/gltoolbox.js.git

it contains the following Modules:

  • GLT.obj.parse: Parses a OBJ file into a convenient vertex array.
  • GLT.loadmanager: fetch files asynchroniously to maximize loading speed.
  • GLT.shader: helps you loadshader.
  • more features will follow.
